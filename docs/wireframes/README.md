# docs/wireframes/

Wireframe HTML tĩnh cho từng màn hình.

## Quy ước

- 1 file = 1 màn hình: `user-list.html`, `user-detail.html`
- Tên file khớp với file trong `docs/screens/`
- Không cần đẹp — chỉ cần đúng layout và structure

## Tôi đọc wireframe như thế nào

Tôi lấy HTML structure để hiểu:
- Vùng nào nằm đâu (header / sidebar / main / panel)
- Thứ tự các element
- Interaction hints (button label, placeholder text)

Tôi KHÔNG lấy từ wireframe:
- Màu sắc (tôi tự apply theo shadcn/ui)
- Font size chính xác
- Spacing chính xác
